# Event Rules PDFs

Please place your event rules PDF files in this directory with the following names:

- `code-arena-rules.pdf` - Rules for Code Arena event
- `clone-it-rules.pdf` - Rules for Clone It event
- `brain-wave-rules.pdf` - Rules for Brain Wave event
- `reel-riddle-rules.pdf` - Rules for Reel Riddle event

These files will be downloadable from the event details popup when users click the "Download Rules" button.
